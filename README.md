# Военный симулятор: Тактика (Project Zeta)

**Жанр:** Пошаговая браузерная стратегия (Turn-based / RTS Economy).
**Технологии:** HTML5 Canvas, CSS3, Vanilla JavaScript (Single file, без внешних зависимостей сборщиков).

### О продукте
**«Военный симулятор: Тактика»** — это элегантная тактическая игра, совмещающая классические пошаговые бои на сетке с экономикой реального времени. 

Игрок выступает в роли Главнокомандующего, управляющего Главным Штабом, который круглосуточно генерирует казенное Золото. На эти средства необходимо вызывать подкрепления (Пехота, Танки, Артиллерия, Авиация) и управлять их позиционированием на поле боя 8×8 клеток. Параллельно с баталиями, бойцы генерируют специальный ресурс — "Науку" (Лампочки), который расходуется на изучение постоянных апгрейдов в Инженерной Ветке: от медицинского снабжения до удлиненных стволов для дальнобойной артиллерии.

### Ключевые особенности
- **Динамичная экономика гибрида:** ходы ваши, а вот прирост золота ежесекундный. Не зевайте, пока ИИ копит ресурсы!
- **Глубокий менеджмент армии:** каждая боевая единица может наступать и стрелять, после чего фаза её хода истощается.
- **Древо Технологий:** перманентные апгрейды (на HP, урон, дальность стрельбы), сохраняющиеся между битвами в localStorage.
- **Самостоятельный ИИ:** гибкий компьютерный противник способен нанимать войска, анализировать дальность хода и методично подбираться к вашему Штабу.
- **Эстетика варгейма:** интерфейс стилизован под современную военную панель с инфографикой, векторными анимациями техники на Canvas и плотным, лаконичным геймплеем.
